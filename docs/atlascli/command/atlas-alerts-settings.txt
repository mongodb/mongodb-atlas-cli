=====================
atlas alerts settings
=====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Manages alerts configuration for your project.

Use this command to list, create, edit, and delete alert configurations.

-------

   :header-rows: 1

     - Type

     - false
