===============
atlas dataLakes
===============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Manage Atlas Data Lakes for your project.

The datalakes command provides access to your project data lakes. You can create, edit, and delete data lakes.
