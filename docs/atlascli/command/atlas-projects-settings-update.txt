==============================
atlas projects settings update
==============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Updates settings of a given project.

Syntax
------

.. code-block::

   atlas projects settings update [options]

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - --disableCollectDatabaseSpecificsStatistics
     - 
     - false
     - Flag that disables the IsCollectDatabaseSpecificsStatisticsEnabled field in project settings.
   * - --disableDataExplorer
     - 
     - false
     - Flag that disables the IsDataExplorerEnabled field in project settings.
   * - --disablePerformanceAdvisor
     - 
     - false
     - Flag that disables the IsPerformanceAdvisorEnabled field in project settings.
   * - --disableRealtimePerformancePanel
     - 
     - false
     - Flag that disables the IsRealtimePerformancePanelEnabled field in project settings.
   * - --disableSchemaAdvisor
     - 
     - false
     - Flag that disables the IsSchemaAdvisorEnabled field in project settings.
   * - --enableCollectDatabaseSpecificsStatistics
     - 
     - false
     - Flag that enables the IsCollectDatabaseSpecificsStatisticsEnabled field in project settings.
   * - --enableDataExplorer
     - 
     - false
     - Flag that enables the IsDataExplorerEnabled field in project settings.
   * - --enablePerformanceAdvisor
     - 
     - false
     - Flag that enables the IsPerformanceAdvisorEnabled field in project settings.
   * - --enableRealtimePerformancePanel
     - 
     - false
     - Flag that enables the IsRealtimePerformancePanelEnabled field in project settings.
   * - --enableSchemaAdvisor
     - 
     - false
     - Flag that enables the IsSchemaAdvisorEnabled field in project settings.
   * - -h, --help
     - 
     - false
     - help for update
   * - -o, --output
     - string
     - false
     - Output format. Valid values are json, json-path, go-template, or go-template-file.
   * - --projectId
     - string
     - false
     - Project ID to use. Overrides the settings in the configuration file or environment variable.
