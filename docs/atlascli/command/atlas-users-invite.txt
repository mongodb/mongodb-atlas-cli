==================
atlas users invite
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Invite a user.

Syntax
------

.. code-block::

   atlas users invite [options]

.. Code end marker, please don't delete this comment

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - --country
     - string
     - false
     - The ISO 3166-1 alpha two-letter country code of the user's country of residence.
   * - --email
     - string
     - true
     - User's email address.
   * - --firstName
     - string
     - true
     - User's first name.
   * - -h, --help
     - 
     - false
     - help for invite
   * - --lastName
     - string
     - true
     - User's last name.
   * - --mobile
     - string
     - false
     - The user's mobile or cell phone number.
   * - --orgRole
     - strings
     - false
     - User's roles  for the associated organization. Valid values are ORG_MEMBER, ORG_READ_ONLY, ORG_BILLING_ADMIN, ORG_GROUP_CREATOR, and ORG_OWNER.
   * - -o, --output
     - string
     - false
     - Output format. Valid values are json, json-path, go-template, or go-template-file.
   * - --password
     - string
     - false
     - User's password.
   * - --projectRole
     - strings
     - false
     - User's roles  for the associated project. Valid Atlas CLI values are GROUP_OWNER, GROUP_READ_ONLY, GROUP_USER_ADMIN, GROUP_BILLING_ADMIN, GROUP_DATA_ACCESS_ADMIN, GROUP_DATA_ACCESS_READ_ONLY, GROUP_DATA_ACCESS_READ_WRITE, GROUP_CHARTS_ADMIN, and GROUP_CLUSTER_MANAGER. Valid MCLI values are all Atlas CLI values and GROUP_AUTOMATION_ADMIN, GROUP_BACKUP_ADMIN, GROUP_MONITORING_ADMIN.
   * - --username
     - string
     - true
     - Username of the user.
