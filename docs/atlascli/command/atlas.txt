.. _atlas:

=====
atlas
=====

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

CLI tool to manage MongoDB Atlas

Use atlas command help for information on a specific command

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -h, --help
     - 
     - false
     - help for atlas
   * - -P, --profile
     - string
     - false
     - Profile to use from your configuration file. To learn about profiles for the Atlas CLI, see https://www.mongodb.com/docs/atlas/cli/stable/atlas-cli-save-connection-settings/. To learn about profiles for MongoCLI, see https://www.mongodb.com/docs/mongocli/stable/configure/configuration-file/.

Examples
--------

.. code-block::

   #
   
.. code-block::

   # Display the help menu for the config command
   atlas config --help
