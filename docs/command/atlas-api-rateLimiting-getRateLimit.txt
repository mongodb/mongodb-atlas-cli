===================================
atlas api rateLimiting getRateLimit
===================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Get one rate limit endpoint set.

The atlas api sub-command is automatically generated from the MongoDB Atlas Admin API and offers full coverage of the Admin API.
Admin API capabilities have their own release lifecycle, which you can check via the provided API endpoint documentation link.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/api/doc/atlas-admin-api-v2/operation/operation-getratelimit.

For more information and examples, see the referenced API documentation linked above.

Syntax
------

.. code-block::
   :caption: Command Syntax

   atlas api rateLimiting getRateLimit [options]
