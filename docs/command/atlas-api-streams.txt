=================
atlas api streams
=================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. blockquote::

   Public Preview: please provide feedback at https://feedback.mongodb.com/forums/930808-atlas-cli

Returns, adds, edits, and removes Streams Instances.

This resource requires your project ID.
