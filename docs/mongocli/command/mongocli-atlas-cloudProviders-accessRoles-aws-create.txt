====================================================
mongocli atlas cloudProviders accessRoles aws create
====================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Create an AWS IAM role.

Role required to use this command. You must authenticate with a user account or an API key that has the Project Owner role.

Syntax
------

.. code-block::
   :caption: Command Syntax

   mongocli atlas cloudProviders accessRoles aws create [options]

.. Code end marker, please don't delete this comment
