=================================================
mongocli cloud-manager liveMigrations link create
=================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Create one new organization link.

Your API Key must have the Organization Owner role to successfully run this command.

Syntax
------

.. code-block::

   mongocli cloud-manager liveMigrations link create [options]

.. Code end marker, please don't delete this comment

-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -h, --help
     - 
     - false
     - help for create
   * - --linkToken
     - string
     - false
     - Link-token generated by Atlas.
   * - --orgId
     - string
     - false
     - Organization ID to use. Overrides the settings in the configuration file or environment variable.
