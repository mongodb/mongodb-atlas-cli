# Test Inventory

#### Fields Description

- Command: the MongoDB Atlas CLI command without `atlas`
- E2E Atlas:
    -  Possible values: `('Y'|'N')`
    -  Indicates if an e2e test for the command is present `('Y')` or not present `('N')` for Atlas
 - Atlas:
     -  Possible values: `('Y'|'N'|' ')`
     -  Indicates if the command supports `('Y')` or not support `('N'|' ')` Atlas
